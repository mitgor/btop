**Step 1: Create `src/btop_events.hpp`**

Create the header with the complete event type definitions and SPSC ring buffer. Follow the exact design from the research document (11-RESEARCH.md "Complete EventQueue Implementation" section). Key details:

```cpp
// src/btop_events.hpp
#pragma once

#include <array>
#include <atomic>
#include <cstdint>
#include <optional>
#include <type_traits>
#include <variant>
```

**Event types** — all in `namespace event {}`:
- `struct TimerTick {}` — main loop timer (Phase 12 use; included now to avoid breaking variant change later)
- `struct Resize {}` — SIGWINCH
- `struct Quit { int exit_code; }` — SIGINT/SIGTERM
- `struct Sleep {}` — SIGTSTP
- `struct Resume {}` — SIGCONT
- `struct Reload {}` — SIGUSR2
- `struct ThreadError {}` — runner exception (msg stored externally in Global::exit_error_msg)

**AppEvent variant:**
```cpp
using AppEvent = std::variant<
    event::TimerTick, event::Resize, event::Quit,
    event::Sleep, event::Resume, event::Reload, event::ThreadError
>;
static_assert(std::is_trivially_copyable_v<AppEvent>,
    "AppEvent must be trivially copyable for signal-safe queue operations");
```

**EventQueue template class** — `template<typename T, std::size_t Capacity>` with C++20 requires clause `(Capacity > 0 && (Capacity & (Capacity - 1)) == 0)` to enforce power-of-2:
- `alignas(64) std::atomic<std::uint32_t> head_{0}` — written by producer (signal handler)
- `alignas(64) std::atomic<std::uint32_t> tail_{0}` — written by consumer (main loop)
- `std::array<T, Capacity> slots_{}`
- `static constexpr std::uint32_t mask = Capacity - 1`
- `bool push(const T& item) noexcept` — relaxed load head, acquire load tail, check full (h - t >= Capacity), write slot, release store head
- `std::optional<T> try_pop() noexcept` — relaxed load tail, acquire load head, check empty (t == h), read slot, release store tail
- `bool empty() const noexcept` — acquire load both, compare

**Global declaration:**
```cpp
namespace Global {
    inline constexpr std::size_t event_queue_capacity = 32;
    // Definition will be in btop.cpp (Plan 02)
    extern EventQueue<AppEvent, event_queue_capacity> event_queue;
}
```
