<objective>
Migrate Config::bools, Config::ints, and Config::strings from unordered_map<string_view, T> to enum-indexed std::array<T, N>, and update all 364 getB/getI/getS call sites across 12 files to use enum keys.

Purpose: Eliminates string hashing overhead on every config access. While Config is not in the per-pixel hot path, it is read at the start of every draw cycle (~116 getB calls in btop_draw.cpp alone). Enum-indexed arrays provide O(1) access with zero hashing, compile-time key safety (typos become compile errors), and consistent cache locality.

Output: Config fully migrated to enum arrays. All call sites use type-safe enum keys.
</objective>

**Part A: Define BoolKey, IntKey, StringKey enums in `src/btop_config.hpp`**

Enumerate all config keys by reading the initialization maps in `btop_config.cpp`. The key names in the enum should match the string names exactly (with hyphens replaced by underscores since C++ identifiers can't have hyphens).

```cpp
enum class BoolKey : size_t {
    theme_background, truecolor, rounded_corners, graph_symbol_cpu_set,
    // ... all ~42 bool keys from btop_config.cpp bools map initialization ...
    COUNT
};

enum class IntKey : size_t {
    update_ms, proc_sorting_col, // ... all ~13 int keys ...
    COUNT
};

enum class StringKey : size_t {
    color_theme, shown_boxes, graph_symbol, // ... all ~28 string keys ...
    COUNT
};
```

**CRITICAL:** Get the EXACT list of keys from the `bools`, `ints`, and `strings` map initializations in `btop_config.cpp`. Every key in those maps must have a corresponding enum value. The enum ordering determines the array index.

Add constexpr name tables for serialization (config file read/write):
```cpp
inline constexpr std::array<std::string_view, static_cast<size_t>(BoolKey::COUNT)> bool_key_names = {
    "theme_background", "truecolor", "rounded_corners", ...
};
// Similarly for int_key_names, string_key_names
```

Add string-to-enum lookup functions for config file parsing:
```cpp
inline std::optional<BoolKey> bool_key_from_name(std::string_view name) {
    for (size_t i = 0; i < bool_key_names.size(); ++i)
        if (bool_key_names[i] == name) return static_cast<BoolKey>(i);
    return std::nullopt;
}
```
