<objective>
Define all typed data structures for the menu pushdown automaton (PDA): 8 frame structs in a `menu` namespace, a `MenuFrameVar` std::variant, a `PDAAction` enum, and a `MenuPDA` class with push/pop/replace/top/empty operations and bg string lifecycle management. The PDA types follow the exact same variant-based state machine pattern already proven in `btop_state.hpp`. Phase 20 is strictly types and tests -- no wiring to existing menu code.
